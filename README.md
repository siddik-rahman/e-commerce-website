# E-Commerce Website  
A simple e-commerce website using **HTML**, **Tailwind CSS** and **Vanilla JavaScript**. It's really simple and easy to use.   
---  
## Technologies Used:  
- **HTML**  
- **Tailwind CSS**  
- **Vanilla JavaScript**  
---  
## Features:  
- Mobile-first responsive design 
- Smooth scrolling  
- Add to cart functionality   
- Adding and removing items from cart  
---  
## How to Use:  
1. **Clone the repository**
``` 
git clone https://github.com/siddik-rahman/E-Commerce-Website.git
```  
2. Open the `index.html` file in your favorite browser.
3. Start Shopping.
